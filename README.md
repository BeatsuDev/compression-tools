# Video compression tool

### Usage:
```
$> ./compress <input_file> <output_file> [--start [[hh:]mm:]ss] [--end [[hh:]mm:]ss]
```

### Requirements:
`ffmpeg` must be installed and accessable via the command line in which you run this program.
